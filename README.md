## Discovery 

### F407

Školní ARM kit běží na desce discovery. Jednou ze zvláštností školního kitu v revizi 1 je 
pin `PE10 (DIR)`, který je použítý pro posílení sběrnice (připojení LCD). Novější revize
ho nepouživají.
